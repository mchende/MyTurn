## 课程配置结构

| Option | Description | Selected |
|--------|-------------|----------|
| 5 个目标项组成整节课 | 内容平铺，流程另算 | |
| 多个轮次，每轮次再挂目标项 | 课堂由固定阶段组成 | ✓ |
| 自由脚本步骤 | 课堂以脚本流为主 | |

**User's choice:** 多个轮次，每轮次再挂目标项  
**Notes:** 课程不是简单素材列表，而是带有课堂阶段结构。

| Option | Description | Selected |
|--------|-------------|----------|
| 只配核心内容 | 目标项保持最小化 | ✓ |
| 再加老师示范话术 | 将示范也配置化 | |
| 再加目标答案/可接受答案 | 将判断材料前置到配置层 | |

**User's choice:** 只配核心内容  
**Notes:** Phase 1 先不把老师话术和答案规则放进配置层。

| Option | Description | Selected |
|--------|-------------|----------|
| 流程写在系统里 | 内容仅提供素材 | ✓ |
| 流程和内容都配置化 | 高灵活度但复杂 | |
| 半配置化 | 少量流程节点可调 | |

**User's choice:** 流程写在系统里  
**Notes:** 课堂推进逻辑属于系统能力，而不是配置能力。

| Option | Description | Selected |
|--------|-------------|----------|
| 固定课堂阶段 | 热身/复述/看图问答/收尾等 | ✓ |
| 重复回合 | 主要按目标项循环 | |
| 自由脚本片段 | 以自由步骤构成课堂 | |

**User's choice:** 固定课堂阶段  
**Notes:** 轮次更像课堂中的自然阶段，而不是技术上的 loop。

| Option | Description | Selected |
|--------|-------------|----------|
| 同一批目标项跨轮次重复 | 逐步提高输出要求 | ✓ |
| 每轮次只用一部分目标项 | 内容拆分使用 | |
| 不同轮次用不同目标项集合 | 轮次之间内容不绑定 | |

**User's choice:** 同一批目标项跨轮次重复  
**Notes:** 同一节课内的训练提升，发生在同一批内容上。

| Option | Description | Selected |
|--------|-------------|----------|
| 一周复用同一底层内容 | 下一周再换新内容 | ✓ |
| 每节课独立配置 | 不做周级复用 | |
| 暂不明确 | 留给后续 planner 决定 | |

**User's choice:** 一周复用同一底层内容  
**Notes:** 用户额外说明“每周重复底层内容，下一周配置新的内容”。
