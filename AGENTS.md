# MyTurn Agent Guide

## Project Snapshot

MyTurn is a web-based children's English speaking classroom MVP focused on recreating the feel of a real online small-group class. The current milestone is a single 15-minute lesson with one teacher, one real child, and one AI classmate, optimized for landscape tablet use.

## Workflow

- Prefer working through GSD phase commands so planning artifacts stay aligned with implementation.
- Next expected command is `$gsd-discuss-phase 1` or `$gsd-plan-phase 1`.
- Keep the MVP focused on classroom feel, not content scale or platform expansion.

## Product Guardrails

- The experience should feel like a short class, not a worksheet or quiz tool.
- Teacher-led turn taking and breathing room before the child's answer are core behaviors, not polish.
- English-first, image-led comprehension is part of the product thesis.
- Lenient answer handling should preserve confidence and pace instead of acting like strict pronunciation scoring.
